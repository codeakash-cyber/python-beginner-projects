# Python Beginner Projects

A collection of small Python programs built while learning core programming concepts.

## Projects
- **calculator.py** — basic arithmetic calculator (add, subtract, multiply, divide)
- **guess_the_country.py** — guessing game with 3 attempts and case-insensitive matching

## Concepts Practiced
- Variables, input/output
- Loops (while, for)
- Conditionals
- Boolean logic
- String methods (`.lower()`)
