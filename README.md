# 🐍 Python Beginner Projects

A collection of beginner-friendly Python programs and mini projects built while learning core Python programming concepts. This repository documents my learning journey through hands-on projects, with each program focusing on improving problem-solving and programming skills.

## 📂 MiniProjects

* **calculator.py** — Performs basic arithmetic operations (addition, subtraction, multiplication, and division).
* **guess_the_country.py** — A country guessing game with three attempts and case-insensitive matching.
* **dice_roller.py** — An interactive dice rolling game that simulates rolling two dice, calculates the total, tracks the number of rolls, and detects doubles and double sixes (Jackpot!).

## 🧠 Concepts Practiced

* Variables and Data Types
* User Input and Output
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Functions
* Boolean Logic
* String Methods (`.lower()`)
* Formatted Strings (f-strings)
* Random Number Generation (`random` module)

This repository serves as a record of my Python learning journey. As I continue learning new concepts, I'll keep adding projects and improving existing ones to strengthen my programming skills.
